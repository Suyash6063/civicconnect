# CivicConnect 🏙️

A next-generation citizen platform designed to modernize urban infrastructure reporting, track municipal action, and build resilient communities. Built specifically to empower citizens and streamline municipal response grids.

## 🚀 Live Deployment
The platform is currently deployed and hosted live via Vercel.

## ✨ Key Features
* **Real-Time Incident Reporting:** Instantly log civic anomalies like potholes, water leaks, and electrical hazards.
* **Interactive GIS Live Map:** A fully interactive spatial dashboard visualizing the health of the neighborhood in real-time.
* **AI Priority Queue:** Automated threat-level classification and algorithmic priority scoring based on severity.
* **Community Hub:** A dedicated forum for citizens to discuss local issues and engage with municipal officials.
* **Analytics Dashboard:** Transparent tracking of municipal department clearance rates and locality hotspots.

## 🛠️ Technology Stack
* **Frontend:** Next.js (React), TypeScript, Tailwind CSS
* **Mapping:** Leaflet.js, React-Leaflet
* **Backend & Database:** Supabase (PostgreSQL)
* **Deployment & Hosting:** Vercel
